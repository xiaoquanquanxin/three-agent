# 关于 Three.js 渲染方式和几何数据存储的确认

> 时间：2026-01-12

---

## 1. Three.js 渲染方式 ✅

**确认：**
- ✅ 相机要 3D（可以旋转、缩放视角）
- ✅ 渲染的图形可以是 2D（平面形状）

**理解：**
- 使用 Three.js 的 3D 相机（PerspectiveCamera 或 OrbitControls）
- 图形本身是 2D 平面（ShapeGeometry、CircleGeometry 等）
- 平面形状平放在 XZ 平面上

---

## 2. 几何数据存储格式 ✅

**确认：**
- ✅ 用一个额外的字段 `vertexList` 存储顶点的 JSON

**示例：**
```json
{
  "id": "square_001",
  "type": "square",
  "vertexList": [
    [7.5, 0, 7.5],
    [12.5, 0, 7.5],
    [12.5, 0, 12.5],
    [7.5, 0, 12.5]
  ],
  "position_x": 10,
  "position_y": 0,
  "position_z": 10
}
```

**理解：**
- `vertexList` 存储所有顶点坐标（JSON 格式）
- 正方形：4 个顶点
- 三角形：3 个顶点
- 圆形：可能存储中心点和半径，或者圆周上的采样点
